# Install
```
sudo curl -L -o /usr/local/bin/scorocode https://github.com/NikSmith/scorocode-cli/raw/0.1.0/scorocode | sudo chmod +x /usr/local/scorocode
```

## Usage

### Fetch project

```
scorocode fetch --appId= --clientKey= --accessKey= --dir=
```

### Publish project
```
scorocode publish --dir=
```
